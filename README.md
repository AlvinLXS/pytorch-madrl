# pytorch-madrl

This project includes PyTorch implementations of various Deep Reinforcement Learning algorithms for both single agent and multi-agent.

- [ ] A2C
- [ ] DQN
- [ ] DDPG

# Requirements

- gym
- python 3.6
- pytorch

# Usage

To train a model:

```
$ python run_a2c.py
```
